Sistema de Controle de Estoque

Projeto simples em Java criado para praticar orientação a objetos, listas, validações e entrada de dados pelo console.

O sistema permite cadastrar produtos, alterar preço, movimentar estoque e listar os produtos cadastrados.

Funcionalidades
Cadastrar produtos
Alterar preço
Adicionar ou remover estoque
Buscar produto por ID
Listar produtos cadastrados
Validar preço e estoque
Como executar

Clone o repositório, abra o projeto em uma IDE Java e execute a classe Program.
